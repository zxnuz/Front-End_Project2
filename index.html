HTML (Hypertext Markup Language)

HTML é a linguagem de marcação padrão para criar páginas da web. Foi proposto por Tim Berners-Lee em 1989 e tornou-se um padrão em 1993 com a publicação do HTML 2.0.

CSS (Cascading Style Sheets)

CSS é uma linguagem de estilo usada para estilizar elementos escritos em uma linguagem de marcação como HTML. Surgiu em 1996 como uma resposta à necessidade de separar a estrutura do documento de sua apresentação.

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e de script comum. Foi criada por Brendan Eich em 1995 enquanto trabalhava na Netscape Communications Corporation.

Python

Python é uma linguagem de programação de alto nível, criada por Guido van Rossum e lançada pela primeira vez em 1991. Ela se destaca por sua sintaxe simples e legibilidade.

Java

Java é uma linguagem de programação orientada a objetos desenvolvida pela Sun Microsystems na década de 1990. Ela foi projetada para ser portável, o que significa que os programas Java podem ser executados em qualquer dispositivo que tenha um interpretador Java.

C++

C++ é uma linguagem de programação de propósito geral criada por Bjarne Stroustrup em 1983. Ela é uma extensão da linguagem de programação C e adiciona suporte para programação orientada a objetos.
